# Tristan Tuico — QA Automation Portfolio

A single-page portfolio site showcasing QA automation work in **Python, pytest, and Robot Framework** — styled around a test-report aesthetic, with the hero built to read like a passing pytest run.

**🔗 Live site:** https://tristantuico.github.io/portfolio/

## About

QA Automation Engineer focused on building test frameworks that are readable, maintainable, and fast enough to run on every commit — not just nightly builds.

## What's on the site

- **Skills** — Python, pytest, Robot Framework, API testing, CI/CD, and general QA tooling, organized as "test modules"
- **Projects** — automation suites built against public practice targets:
  - *UI Regression Suite — SauceDemo (Swag Labs)*: keyword-driven Robot Framework suite covering login, cart, sort, and checkout flows
  - *API Test Suite — Restful-Booker*: pytest suite with full CRUD coverage, schema validation, and auth token testing
- **Experience** — professional QA automation background
- Full write-ups for each project are available at [`writeups.html`](writeups.html)

## Tech

Built as static HTML/CSS with no build step or dependencies — easy to fork, edit, and deploy on GitHub Pages.

## Files

- `index.html` — the main site
- `writeups.html` — detailed case studies for each project
- `README.md` — this file

## Local editing

Clone the repo and open `index.html` directly in a browser — no server or build tools required.

```bash
git clone https://github.com/tristantuico/portfolio.git
cd portfolio
open index.html
```
